# Pharma Tech SaaS - نظام إدارة الصيدليات الذكي 🏥💊

نظام سحابي متكامل لإدارة الصيدليات، المبيعات، المخزون، وعلاقات المرضى، مصمم خصيصاً للسوق المصري مع واجهة عربية كاملة ودعم RTL.

## ✨ المميزات الرئيسية

### 🏪 **نقطة بيع متطورة (POS)**
- دعم كامل لماسحات الباركود (Barcode Scanners)
- نظام FIFO آلي لإدارة المخزون
- دعم الكميات الكسرية للأدوية
- طباعة فواتير احترافية مع QR Code
- إشعارات واتساب تلقائية للمرضى

### 📊 **إدارة المخزون الذكية**
- تتبع الأدوية قريبة الانتهاء
- تنبيهات النواقص الآلية
- نظام إعادة التموين الذكي
- تحليل المخزون الراكد (Dead Stock)
- دعم الكميات الكسرية

### 👥 **إدارة المرضى (CRM)**
- سجلات طبية كاملة
- ملاحظات الحساسية والأمراض المزمنة
- تنبيهات إعادة ملء الوصفات
- تاريخ المشتريات السابقة
- إشعارات واتساب تلقائية

### 📈 **التقارير والتحليلات**
- تقارير مبيعات يومية وأسبوعية وشهرية
- مؤشرات أداء الموظفين (KPIs)
- تحليل هامش الربح
- تقارير المخزون والانتهاء
- **نظام إدارة الشفتات النقدية الجديد**

### 🔐 **نظام الشفتات النقدية** (ميزة جديدة)
- فتح وإغلاق الشفتات النقدية
- تتبع الفروق النقدية (Overage/Shortage)
- سجل المعاملات الملغاة (Void Transactions)
- تدقيق وتحقق المديرين
- تقارير التقلص (Shrinkage Reports)

### 🎨 **واجهة مستخدم حديثة**
- تصميم عربي كامل (RTL)
- وضع ليلي/نهاري
- تجربة مستخدم سلسة وسريعة
- تصميم متجاوب لجميع الأجهزة
- ألوان مخصصة للصيدليات

## 🚀 البدء السريع

### المتطلبات الأساسية
- Node.js 18+ 
- npm أو yarn أو pnpm
- حساب Supabase (مجاني)

### خطوات التثبيت

1. **نسخ المستودع**
   ```bash
   git clone <repository-url>
   cd pharma
   ```

2. **تثبيت التبعيات**
   ```bash
   npm install
   # أو
   yarn install
   # أو  
   pnpm install
   ```

3. **إعداد متغيرات البيئة**
   أنشئ ملف `.env.local` في المجلد الرئيسي وأضف:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **تشغيل خادم التطوير**
   ```bash
   npm run dev
   # أو
   yarn dev
   # أو
   pnpm dev
   ```

5. **فتح المتصفح**
   انتقل إلى `http://localhost:3000`

### إعداد قاعدة البيانات

1. سجل الدخول إلى [Supabase Dashboard](https://supabase.com/dashboard)
2. أنشئ مشروع جديد أو استخدم مشروع موجود
3. انسخ `URL` و `anon key` إلى ملف `.env.local`
4. (اختياري) قم بتشغيل سكربت إعداد قاعدة البيانات من مجلد `scripts/`

## 🔧 الميزات الجديدة والمحسنة

### نظام إدارة الشفتات النقدية
تم إضافة نظام كامل لإدارة الشفتات النقدية يتضمن:
- **فتح الشفتات**: تسجيل الرصيد الافتتاحي والملاحظات
- **إغلاق الشفتات**: حساب الفروق النقدية تلقائياً
- **تتبع المعاملات الملغاة**: مراقبة عمليات الإلغاء للتحكم في التقلص
- **تقارير التقلص**: تحليل الفروق النقدية وأنماط الإلغاء
- **تحقق المديرين**: موافقة المدير على الشفتات ذات الفروق الكبيرة

### تحسينات واجهة المستخدم
- **تصميم أزرار محسن**: أزرار ذات تأثيرات ظل وتدرجات ألوان
- **أيقونات توضيحية**: استخدام مكتبة Lucide React للأيقونات
- **تحسين تجربة النماذج**: تحقق في الوقت الفعلي وتصميم أفضل
- **تحسين الجداول**: تصميم أكثر وضوحاً مع ترميز الألوان للحالات
- **رسائل تفاعلية**: تنبيهات نجاح/خطأ محسنة

### تحسينات الأمان
- **Server Actions**: تحويل جميع عمليات قاعدة البيانات إلى Server Actions
- **تحقق Zod**: تحقق من صحة جميع بيانات الإدخال
- **عزل البيانات**: ضمان وصول المستخدمين فقط لبيانات صيدليتهم
- **تدقيق السجلات**: تسجيل جميع الإجراءات المهمة
