# ⚡ الدليل السريع للبدء

## خطوات البدء في 5 دقائق

### 1. تثبيت التبعيات
```bash
npm install
```

### 2. إعداد متغيرات البيئة
أنشئ ملف `.env.local` في المجلد الرئيسي وأضف:
```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here
```

للحصول على هذه القيم:
1. سجل الدخول إلى [Supabase Dashboard](https://supabase.com/dashboard)
2. أنشئ مشروع جديد أو اختر مشروع موجود
3. انتقل إلى Project Settings → API
4. انسخ `URL` و `anon public key`

### 3. تشغيل التطبيق
```bash
npm run dev
```

### 4. فتح المتصفح
انتقل إلى: [http://localhost:3000](http://localhost:3000)

### 5. تسجيل الدخول
- استخدم بيانات اختبار أو أنشئ حساب جديد
- الصيدلية الأولى التي تنشئها ستصبح الصيدلية الرئيسية

## 🔧 إعداد قاعدة البيانات (اختياري)

إذا كنت تريد بيانات تجريبية:

### تشغيل سكربت البذور
```bash
# تشغيل سكربت Python لملء قاعدة البيانات بالأدوية
python seed_db.py

# أو تشغيل سكربت Node.js للبيانات التجريبية
node scratch/seed.js
```

### تطبيق مخطط الشفتات النقدية
1. افتح Supabase SQL Editor
2. انسخ محتوى `scripts/shift-registers-schema.sql`
3. قم بتنفيذ SQL لإنشاء جداول الشفتات

## 🚨 استكشاف الأخطاء وإصلاحها

### المشكلة: خطأ في اتصال Supabase
**الحل**: تأكد من:
1. متغيرات البيئة صحيحة في `.env.local`
2. المشروع نشط في Supabase Dashboard
3. الاتصال بالإنترنت يعمل

### المشكلة: لا تظهر البيانات
**الحل**: 
1. تحقق من سياسات RLS في Supabase
2. تأكد من وجود بيانات في الجداول
3. جرب تسجيل الخروج والدخول مرة أخرى

### المشكلة: أخطاء TypeScript
**الحل**:
```bash
# تثبيت التبعيات المفقودة
npm install --save-dev @types/node @types/react @types/react-dom

# أو إعادة بناء ملفات TypeScript
rm -rf .next && npm run build
```
